# Texture-Pack-Graying
A grayscaling tool for Minecraft 1.8.9 texture packs.

------------

#### Usage
* Put your texture packs in `./input/` folder, can be folders or `.zip` files
* Run `./main.py` anyway 
* Check `./output/` folder

Modify `config.yml` to discover your own unique visual effect.
